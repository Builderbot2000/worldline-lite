# Rival Nation Template — DM-Run Powers

A **condensed** sheet for a power the World-Sim controls. Deliberately lighter than the player's
`nation-sheet.md` so World-Sim can run several at once. Track exact numbers only where they drive
interaction with the player (forces that might fight you, trade you rely on); everything else uses
**bands**. Copy one per power to `runs/<run-name>/rivals/<power>.md`.

> Bands: **Negligible · Low · Moderate · High · Dominant** (relative to the run's scale).
